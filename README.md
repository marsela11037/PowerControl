# Инструкция к ПО: Power Control

## 1. Назначение
`Power Control` - это Windows-приложение, которое запускает локальный HTTP-сервер и позволяет с телефона в одной сети:
- выключать ПК;
- переводить ПК в сон;
- отправлять Wake-on-LAN (если настроен MAC и broadcast).

## 2. Требования
- ОС: Windows 10/11.
- Python 3.9+ (если запуск из исходников).
- Телефон и ПК должны быть в одной локальной сети (обычно один Wi-Fi).

## 3. Быстрый запуск
### Вариант A: запуск EXE
1. Запустите `build_simple.bat` (сборка).
2. Дождитесь создания `dist/PowerControl.exe`.
3. Запустите `PowerControl.exe`.

### Вариант B: запуск из исходников
1. В папке проекта выполните:
   - `pip install -r requirements.txt`
2. Запустите:
   - `python power_app.py`

## 4. Как пользоваться
1. После запуска приложение автоматически поднимает сервер.
2. В окне отображаются:
   - `IP:порт`;
   - токен доступа;
   - URL и QR-код для телефона.
3. Откройте URL на телефоне или сканируйте QR.
4. На телефоне используйте команды управления питанием.

## 5. Настройка безопасности
- Обязательно измените токен по умолчанию `marsela` на свой.
- Используйте только в доверенной локальной сети.
- Не пробрасывайте порт в интернет без дополнительной защиты.

## 6. Настройка порта
- Порт можно изменить в интерфейсе (кнопка "изменить порт").
- Допустимый диапазон в приложении: `1024-65535`.
- После изменения порт сохраняется в пользовательский конфиг.

## 7. Автозапуск с Windows
- В интерфейсе есть переключатель "Запускать с Windows".
- При включении приложение добавляет запись в реестр `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`.
- Используется флаг `--minimized` для старта в трее.

## 8. Работа в системном трее
- При закрытии окна можно свернуть приложение в трей.

## 9. REST API (для интеграций)
### GET
- `GET /health` -> проверка доступности сервера.
- `GET /shutdown?token=<TOKEN>` -> выключение ПК.
- `GET /sleep?token=<TOKEN>` -> сон ПК.

### POST (JSON)
- `POST /api/shutdown` `{ "token": "<TOKEN>" }`
- `POST /api/sleep` `{ "token": "<TOKEN>" }`
- `POST /api/wake` `{ "token": "<TOKEN>", "pc_id": "my_pc" }`

## 10. Типовые проблемы
- Телефон не открывает URL:
  - проверьте, что устройства в одной сети;
  - проверьте брандмауэр Windows и порт;
  - используйте локальный IP ПК, не `localhost`.
- Сервер не стартует:
  - проверьте, не занят ли порт другим процессом.
- Wake-on-LAN не работает:
  - проверьте BIOS/UEFI и настройки сетевого адаптера.
