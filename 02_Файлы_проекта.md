# Файлы проекта

## Ключевые файлы
- `power_app.py` - основной файл приложения: GUI (Tkinter), HTTP-сервер, API, трей, автозапуск, QR.
- `templates/index.html` - веб-интерфейс, который открывается на телефоне.
- `static/manifest.json` - PWA-манифест.
- `static/sw.js` - Service Worker для PWA.
- `static/icon-192.png`, `static/icon-512.png` - иконки PWA.
- `requirements.txt` - Python-зависимости.
- `README.md` - пользовательское описание проекта.

## Файлы сборки
- `build_simple.bat` - быстрый bat-скрипт сборки в EXE через PyInstaller.
- `build_exe.py` - Python-скрипт сборки EXE с проверкой зависимостей.
- `PowerControl.spec` - конфигурация PyInstaller.
- `build/` - временные и служебные файлы сборки PyInstaller.

## Ресурсы
- `app_icon.ico`, `app_icon.png` - иконки приложения.

## Конфигурация во время работы
- `%USERPROFILE%\\.power_control_config.json` (создается автоматически) - сохраняет:
  - токен;
  - порт.

## Логическая структура `power_app.py`
- Блок конфигурации (`PORT`, `SECRET_TOKEN`, `WOL_COMPUTERS`).
- HTTP-обработчик `PowerHandler` с `GET` и `POST` endpoint'ами.
- GUI-класс `PowerControlApp`:
  - загрузка/сохранение конфигурации;
  - генерация QR;
  - старт/стоп сервера;
  - сворачивание в трей;
  - настройка автозапуска.
