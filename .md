# Power Control (LAN) - Версия 2.0

Выключайте или переводите Windows‑ПК в сон с телефона (в одной Wi‑Fi/локальной сети).

## ✨ Возможности

- 🎨 **Красивое GUI приложение** - современный интерфейс вместо консоли
- 📱 **PWA для телефона** - установка на домашний экран как приложение
- 🖥️ **Системный трей** - работа в фоне с иконкой в трее
- ⚡ **Быстрый запуск** - просто запустите start_server.bat

---

## 🚀 Быстрый старт

### Вариант А: Готовый EXE (рекомендуется)

1. **Скомпилируйте приложение:**
   - Дважды кликните `build_simple.bat`
   - Подождите завершения компиляции

2. **Запустите:**
   - Найдите `dist/PowerControl.exe`
   - Дважды кликните - сервер запустится автоматически!
   - Скопируйте URL из окна приложения
   - Откройте URL на телефоне

3. **Готово!** Можете скопировать PowerControl.exe куда угодно и запускать без Python.

### Вариант Б: Запуск из исходников

1. Установите зависимости:
```powershell
pip install -r requirements.txt
python create_icons.py
```

2. Запустите:
```powershell
python power_app.py
```
Или дважды кликните `start_server.bat`

## ⚙️ Настройка

### Токен и порт

По умолчанию:
- **Токен**: `marsela`
- **Порт**: `8765`

Изменить можно в `power_server.py` или через переменные окружения:
```powershell
$env:POWER_TOKEN="ваш_сложный_токен"
$env:POWER_PORT="8765"
python power_server_gui.py
```

### Wake-on-LAN (включение ПК)

Чтобы **включать** ПК удаленно:

1. **В BIOS:** включите **Wake on LAN** / **Power On By PCI‑E**
2. **В Windows:** Свойства сетевого адаптера → Управление питанием → "Разрешить этому устройству выводить компьютер из ждущего режима"
3. **В коде:** в `power_server.py` заполните `WOL_COMPUTERS`:

```python
WOL_COMPUTERS = {
    "my_pc": {
        "name": "Мой ПК",
        "mac": "XX-XX-XX-XX-XX-XX",  # ipconfig /all → Физический адрес
        "broadcast": "192.168.1.255",  # Обычно .255 в конце
        "port": 9,
    }
}
```

### Автозапуск при входе в Windows

```powershell
python install_autostart.py
```

Удалить из автозагрузки:
```powershell
python install_autostart.py remove
```

## 🔧 Решение проблем

### Телефон не подключается

1. Убедитесь, что оба устройства в **одной Wi-Fi сети**
2. Отключите мобильный интернет на телефоне
3. Разрешите порт в брандмауэре:
```powershell
netsh advfirewall firewall add rule name="Power Server" dir=in action=allow protocol=TCP localport=8765
```
4. Используйте IP адрес (не localhost)

### GUI не запускается

Установите зависимости:
```powershell
pip install pystray Pillow
```

### Не работает Wake-on-LAN

- Проверьте настройки BIOS
- Проверьте настройки сетевого адаптера в Windows
- Убедитесь, что MAC адрес и broadcast указаны правильно
